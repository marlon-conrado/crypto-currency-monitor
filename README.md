## Setup local environment

#### Node version

- ^14.15.3

#### Install dependencies

- `npm install` o `yarn`

#### Mandatory environment variables

Must create file .env in the root of the project

- DB_HOST
- DB_PASS
- DB_USER_NAME
- DB_NAME
- NODE_ENV
- DB_PORT

#### Run app

- `npm run dev` or `yarn dev`
